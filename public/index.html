Magic Points API

Backend service for the Hogwarts Inequality Web Application

API Endpoints

GET /api/user/magic-points

Get the current user's magic points

Authentication: Required

POST /api/user/magic-points

Update the user's magic points

Authentication: Required

Body: { "magicPoints": number }

POST /api/user/magic-points/sync

Sync offline operations

Authentication: Required

Body: { "operations": array }
